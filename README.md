# Muhammad Imran Portfolio - Angular Version

This is an Angular conversion of the original HTML portfolio website.

## Features

- ✅ Angular 21 standalone components
- ✅ Responsive design with Tailwind CSS
- ✅ GSAP animations and scroll effects
- ✅ Particle background canvas
- ✅ Smooth scroll navigation
- ✅ Mobile-friendly navigation
- ✅ All original sections converted

## Sections

1. **Hero** - Introduction with animated typewriter and profile card
2. **About** - Personal information and stats
3. **Services** - Service offerings with hover effects
4. **Skills** - Tech stack showcase
5. **Projects** - Portfolio projects
6. **Experience** - Work history timeline
7. **Contact** - Contact form

## Installation & Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
ng serve
```

3. Open browser to `http://localhost:4200`

## Build for Production

```bash
ng build --configuration production
```

The build artifacts will be stored in the `dist/` directory.
