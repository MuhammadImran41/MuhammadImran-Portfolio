## ❓ Need Help?

### Common Issues

**Port already in use?**
```bash
ng serve --port 4300
```

**Animations not working?**
- Refresh browser (Ctrl+F5)
- Check browser console for errors

**Build errors?**
```bash
npm install
ng build --configuration development
```

## 🌐 Deploy Your Portfolio

After building, deploy to:

**Easiest Options:**
1. **Vercel** - Best for Angular
   ```bash
   npm install -g vercel
   vercel deploy
   ```

2. **Netlify** - Drag and drop `dist` folder

3. **GitHub Pages** - Use angular-cli-ghpages

See **[QUICK_START.md](QUICK_START.md)** for detailed deployment instructions.
